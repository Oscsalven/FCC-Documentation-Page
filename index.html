Introduction

This is a documentation page where Javascript loops will be taught. This loops are very useful for performing some task over and over again, as long as a specific condition is true, so when it changes to false, the loop stops. The loops coverered are:

  • for
  • forEach
  • while
  • do .... while
  • for ... in
  • break
Let's take a look.

For Loop

The for loop helps us to perform one tasks several times, each iteration will use different values, until certain condition, set by us, is false.

The for loop is created in the following way:

for(x, y, z) {
// do stuff
}

Where:

  • x is the starting condition, this will run only once
  • y is the condition that must be met every time the code runs
  • z represents how x will change once the code has ran

ForEach Loop

The forEach loop works with an array. This method will loop through each element in an array and call a function on it.

The sintax looks like this:

array.forEach(element => {// do stuff})

While Loop

The while loop will run some code every time a condition we set is true. The while Loop will verify this before running the code. The sintax is the following:

while(condition) {
//do stuff
}

Do While Loop

This a variant of the while loop. In this case, the code will run and once it's done, the loop will verify if our condition is still true.

do {
// do stuff
}
while (condition);

For In Loop

This loop iterates through the elements of an object or an array through a key, which will access the value of each element. The sintax is the same for both objects and arrays.

Let's look at the sintax.

for (variable/key in array/obeject) {
//do stuff
}
